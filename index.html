📋 วิธี Setup (Step-by-step)

  1. สร้าง Google Sheet ตาม format ด้านบน (1 Tab ต่อ 1 เดือน)
  2. ไปที่ File → Share → Publish to web
  3. เลือก Tab เดือนนั้น → Format = CSV → กด Publish
  4. Copy URL ที่ได้ → กลับมาที่ Dashboard → เลือก Tab เดือน (ด้านบน)
  5. วาง URL ในช่องด้านบน → กด "โหลด & บันทึก URL"
  6. URL จะถูกบันทึกในเครื่องถาวร — Auto-refresh ทุก 5 นาที

⚠ ต้อง Publish to web ก่อนเท่านั้น — URL ปกติของ Google Sheets ใช้ไม่ได้เพราะ CORS

✓ URL ที่ถูกต้องจะขึ้นต้นด้วย https://docs.google.com/spreadsheets/d/... และลงท้ายด้วย &output=csv
