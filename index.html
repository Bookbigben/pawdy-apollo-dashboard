📲 แชร์ข้อมูลข้ามเครื่อง / GitHub Pages

ข้อมูลถูกบันทึกใน browser นี้เท่านั้น — เลือกวิธีด้านล่างเพื่อใช้งานบนเครื่องอื่นหรือเผยแพร่ทางเว็บ

🚀 GitHub Pages — เปิด URL แล้วเห็นข้อมูลทันที
  1. ซิงค์ข้อมูลให้ครบก่อน (↻ โหลดทุกเดือน)
  2. กดปุ่ม 💾 ดาวน์โหลดไฟล์ ด้านล่าง
  3. Rename ไฟล์ที่ได้เป็น index.html
  4. Push ขึ้น GitHub repo (main branch)
  5. เปิด GitHub Pages URL → เห็นข้อมูลทันที ✅

💡 ข้อมูลฝังอยู่ในไฟล์ — ไม่ต้องซิงค์ใหม่ทุกครั้ง · เมื่อข้อมูลเปลี่ยน ทำซ้ำขั้นตอน 1-4

📋 เครื่องอื่น / browser อื่น — Export / Import JSON

Export URL ทั้งหมดเป็น JSON → Copy → เปิดไฟล์บนเครื่องอื่น → Paste → Import → ↻ โหลด

📋 วิธี Setup (Step-by-step)

  1. เปิด Google Sheets → File → Share → Publish to web
  2. เลือก Tab ของเดือนนั้น (เช่น ม.ค., ก.พ., …) → Format = CSV → กด Publish
  3. Copy URL ที่ได้ → วางในช่อง 📊 Channel ของเดือนนั้นด้านบน
  4. ทำซ้ำสำหรับ Tab Live (🎥) และ Affiliate (🛒) ถ้ามี
  5. กด ↻ โหลด ที่เดือนนั้น หรือ ↻ โหลดทุกเดือน เพื่ออัปเดตทีเดียว
  6. URL บันทึกในเครื่องถาวร — Auto-refresh ดึงข้อมูลใหม่ทุกเดือนอัตโนมัติ

⚠ ต้อง Publish to web ก่อนเท่านั้น — URL ปกติของ Google Sheets ใช้ไม่ได้เพราะ CORS

✓ URL ที่ถูกต้องจะขึ้นต้นด้วย https://docs.google.com/spreadsheets/d/... และลงท้ายด้วย &output=csv
